# Processing で名刺を作ろう！

[Processing.py](https://py.processing.org/)で名刺を作成するためのプログラムです。

## 動作環境（例）

- OS: Windows 11
- Processing: 4.0 beta 8 (REV 1283)
- Python Mode for Processing 4: 4001
